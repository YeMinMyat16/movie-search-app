## Features
- Search for any movie or TV series using the OMDb API.
- View movie posters, release years, and IMDB ratings.
- Instantly visit the IMDB page for full details with a single click.
- Modern, responsive, and beautiful UI built with Tailwind CSS.

## Getting Started

1. Get your free API key at [omdbapi.com](http://www.omdbapi.com/)
2. Set up your OMDb API key in a `.env.local` file (do NOT commit this file to git):
   ```env
   VITE_OMDB_API_KEY=your_api_key_here
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
